# Corona HTML5 Node Kit

__Build Corona HTML5 plugins using NodeJS, NPM, and Modern Javascript.__

## Installation

### NodeJS

Install __[NodeJS](https://nodejs.org)__ version 8 or better for _macOS_ or _Windows_.

### Download Kit

Download the _Corona HTML5 Node Kit_ by __[clicking here](https://github.com/develephant/corona-html5-node-kit/archive/master.zip)__.

## CLI Commands

### init

```
coronakit init --app DemoApp --plugin myplugin --id com.me
```

### compile

```
coronakit compile
```

### build

```
coronakit build
```

### watch

```
coronakit watch
```

### debug

```
coronakit debug
```
